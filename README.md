# Project Title:
To Cook or Not to Cook

Project Description:
Using the To Cook or Not to Cook app, you can either search for recipes to cook from home or restaurants nearby based off of the type of food you are craving.

USER STORY:
AS A person hungry for my favorite food 
I WANT to find recipes OR local restaurants
SO THAT I can cook from my own home or order from a restaurant
